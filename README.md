# Coffee Shop Sales Dashboard - Excel

## Project Description

Analyzed sales, revenue, and profit trends using an interactive Excel dashboard. Identified top-performing products, cities, and peak sales periods.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards

## Key Metrics

* Total Transactions: 9,971
* Total Revenue: $46,374
* Total Profit: $16,231
* Total Quantity Sold: 14,117

## Key Insights

* Tea Beverages generated the highest revenue.
* Friday recorded the highest number of transactions.
* Revenue was distributed across multiple cities.
* Beverage categories contributed the majority of sales.
